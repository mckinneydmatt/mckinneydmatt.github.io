My Work

PetPickr

PetPickr was my final project for Eleven Fifty Academy. I created a C# MVC web application to find adoptable pets. My project included three levels of roles for users, giving more or less access in each role. I'm currently working on expanding it to include public APIs from Google Maps and PetFinder.

MoneyManager

This was a group project with two of my classmates, showing our knowledge of REST APIs. We created a money management app, allowing users to input their account balances and latest expenses. It was run primarily through Postman.

CSS Creature

This project was made entirely through CSS. The object was to create a creature of any kind, using only CSS object manipulation. I went with the alien from the Toy Story franchise. Getting each of the alien's fingers exactly where I wanted them was a real challenge.
